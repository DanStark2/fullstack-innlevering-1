# Fullstack-prosjekt – Restaurant Tangen Torv

Dette prosjektet er en fullstack-løsning for restauranten Tangen Torv. Løsningen skal vise informasjon om restauranten, meny, åpningstider og kontaktinformasjon. Det skal også være mulig å sende inn bordbestillinger. Ansatte får en admin-side hvor de kan se bestillinger.  
Prosjektet består av tre hovedmoduler: **Database**, **Webside** og **Backend**.

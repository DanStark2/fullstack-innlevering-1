# Restaurant booking (Fullstack)

- Docker Desktop: 4.50.0
- PostgreSQL: 18.1
- Node.js (LTS): v24.12.0
- Express: 5.2.1

## Modules
1. Database (PostgreSQL + Docker)
2. Frontend (HTML/CSS/JS with hardcoded data)
3. Backend (Node/Express API + database connection) *(after Christmas)*

## Project structure
- /database -> schema.sql + queries.sql
- /frontend -> static pages
- /backend -> API (later)
